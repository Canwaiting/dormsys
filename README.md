## 项目介绍
所用技术栈
+ 简单javaweb知识
+ bootstrap(用于美化)
+ mysql数据库

作者开发环境
> mysql  Ver 8.0.26 for Linux on x86_64
Tomcat 9.0.53
java11

## 快速使用
克隆当前项目并把项目中的数据库文件导入你的数据库即可
(数据库文件在database中,dormsys.sql为建表语句,dormsys_insert.sql为插入数据语句)
